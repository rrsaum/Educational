Comments From Learner Elements

The Comments From Learner Data Model Element is made up of three elements. These elements can be used to track different characteristics about a specific comment:

  • Comment - The textual comment or annotation, provided by the learner, associated with the SCO.
  • Location - The point in a SCO at which the comment applies (if there is no location then the comment is applicable to the whole SCO).
  • Timestamp - The point in time when the comment was created or changed.
