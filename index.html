Managing Files

Main Objectives

» Create a folder in the Manage Files area
» Upload a file in the Manage Files area
» Zip a file in the Manage Files area
» Create a topic in the Manage Files area


Logging into D2L

  1. To manage files you must be logged into Desire2Learn
  2. After logging into Desire2Learn, select your course from the Homepage
  3. Select the Edit Course link from the Desire2Learn toolbar.
  4. Select the Files link
