Managing Files

The New Tool

Play Audio
Use the New link to create folders for organizing your content. For example you could create these folders: Assignments, Lectures, and Projects.
Using the New Tool
  1. Select the New link.
  2. The New Folder dialog appears
  3. Enter the name desired for your new folder
  4. Select the OK button

Upload Tool

Play Audio
Use the upload link to post content into your course. This allows you to post homework, lecture notes and other documents into your course.
Using the Upload Tool
  1. Select the Upload link
  2. Browse for the file you wish to upload
  3. Select the Upload button

The Zip Tool

Play Audio
Use the Zip tool to package your course documents and download them to your computer.
Using the Zip Tool
  1. select the file(s) or folder(s) you wish to download
  2. Select the Zip Files link
  3. select the location to save your zip file to
  4. Select the Save button

The Create Topics Tool

Play Audio
Use the Create Topics Tool to make your material available to students.
Using the Create Topics Tool
  1. Select the files you wish to create topics from
  2. Select the Create Topics link
  3. Select the Parent Mode for the topic(s)
  4. If desired change the title for the topic(s)
  5. Select the Save button
